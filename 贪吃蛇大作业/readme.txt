游玩方法说明：

对于大部分模式，
玩家控制的是绿蛇，方向键为w、a、s、d。
每吃到一个果实加2分。
撞到任何障碍物（在部分模式撞墙游戏不结束）扣10分。
分数最低为0分，上不封顶。

在体能模式下，每走一步耗费一点体能，吃到果实增加三十体能
体能上限100，多加无益

在AI模式与双人对战下，出现第二条蛇，颜色为紫色。
绿蛇的控制仍为w、a、s、d，
如果是双人模式，紫蛇的控制为i、j、k、l。（老街机玩家了）

其中
石头图标表示“硬墙”
红色的苹果表示“果实”
酒瓶图标表示“软墙”
核辐射图标表示“毒果实”
橙色的地块为加速区
蓝色的地块为减速区
